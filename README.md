# EduCraft AI v2 🎓🤖

**EduCraft AI** est une plateforme innovante conçue pour accompagner les éducateurs dans l'automatisation et la création de supports pédagogiques de haute qualité. Grâce à l'intelligence artificielle, transformez vos idées en contenus structurés en quelques clics.

## 🚀 Fonctionnalités
- **Rédaction automatisée :** Génération de cours et de contenus pédagogiques.
- **Générateur de Quiz :** Création instantanée d'évaluations interactives.
- **Fiches de révision :** Synthèse automatique des points clés.
- **Interface Intelligente :** Une UI fluide conçue pour une expérience utilisateur optimale.

## 🛠️ Stack Technique
- **Frontend :** HTML5, CSS3 (Design moderne et responsive), JavaScript (Vanilla).
- **Backend / API :** Intégration d'APIs IA via `api-client.js`.
- **Déploiement :** Configuré pour [Vercel](https://vercel.com).
